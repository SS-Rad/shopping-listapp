# 🛒 쇼핑 리스트 앱

바닐라 JavaScript로 만든 간단한 쇼핑 리스트 웹앱입니다. 별도의 빌드나 서버 없이 브라우저에서 바로 동작합니다.

## 기능

- ✅ 항목 추가 / 삭제
- ✅ 클릭으로 완료(체크) 토글
- ✅ 완료 항목 한 번에 비우기
- ✅ `localStorage`에 자동 저장 (새로고침해도 유지)
- ✅ 반응형 모바일 친화 디자인

## 사용법

`shopping-list.html` 파일을 브라우저에서 열기만 하면 됩니다.

```
shopping-list.html  →  더블클릭 또는 브라우저로 열기
```

## 기술 스택

- HTML / CSS / Vanilla JavaScript
- 외부 의존성 없음, 단일 파일
